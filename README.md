# clock-angles-piped
Code Exercise

This script outputs matching clock angles between RVA and Chicago.

> clocks.js

```javascript
│    0    │ 0  │ 30 │ 165 │ 'h: 23 m: 30 a: 165' │ 
│    1    │ 6  │ 30 │ 15  │  'h: 5 m: 30 a: 15'  │ 
│    2    │ 12 │ 30 │ 165 │ 'h: 11 m: 30 a: 165' │ 
│    3    │ 18 │ 30 │ 15  │ 'h: 17 m: 30 a: 15'  │

```
